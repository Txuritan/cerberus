Grid

Grids use two tags, x-grid and x-column. x-grid is the parent tag and is used as a container. x-column is a container and is used to control the size of its children.

Code
<x-grid>
  <x-column size="3"></x-column>
  <x-column size="6">
    <!-- This docs contents go here -->
  </x-column>
  <x-column size="3"></x-column>
</x-grid>
Output

Navbar

Navbars have two states: Primary and Secondary. Neither do anything to change abilities and only change looks.

Also note that Grids can be used inside Navbars, this page is a great example of that.

Primary Navbar
Code
<x-nav primary>
  <x-nav-section grow>
    <x-nav-item brand>
      <a href="#"> Cerberus</a>
    </x-nav-item>
  </x-nav-section>
  <x-nav-section>
    <x-nav-item>
      <a href="#navbar"> Navbar</a>
    </x-nav-item>
  </x-nav-section>
</x-nav>
Output
Cerberus Navbar
Secondary Navbar
Code
<x-nav secondary>
  <x-nav-section grow>
    <x-nav-item brand>
      <a href="#"> Cerberus</a>
    </x-nav-item>
  </x-nav-section>
  <x-nav-section>
    <x-nav-item>
      <a href="#navbar"> Navbar</a>
    </x-nav-item>
  </x-nav-section>
</x-nav>
Output
